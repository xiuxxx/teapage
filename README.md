# Tea Brand Site

This is a lightweight static brand website that can be deployed directly to Vercel.

## Files

- `index.html`: homepage structure
- `styles.css`: visual design and responsive layout
- `main.js`: mobile navigation and footer year
- `favicon.svg`: site icon
- `vercel.json`: Vercel deployment config

## Quick Deploy

1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Keep the project as a static site. If Vercel asks for a framework preset, choose `Other`.
4. Leave build settings empty:
   - Build Command: empty
   - Output Directory: empty
5. Deploy and bind your custom domain.

## Before Going Live

Replace the placeholder content in `index.html`:

- brand name
- email and phone number
- company address
- product names and descriptions
- business introduction text
